# Build Army List Workflow

## Purpose
Interactive step-by-step army list construction with validation, meta analysis, and competitive optimization.

## Workflow Steps
1. Select faction
2. Choose detachment and game mode
3. Add units (HQ, Troops, Elites, etc.)
4. Validate points and legality
5. Analyze synergies and meta viability
6. Generate final list with meta stats

## Inputs
- User faction preference
- Points limit
- Game mode (Matched Play, Crusade, Open Play)

## Outputs
- Legal, point-accurate army list
- Meta analysis and tournament stats
- Synergy recommendations
- Saved list to tacticus-sidecar
