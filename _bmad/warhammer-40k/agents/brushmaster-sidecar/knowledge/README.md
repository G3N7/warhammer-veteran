# Brushmaster Knowledge Base

Painting techniques and color theory for quick reference.

## Structure

```
knowledge/
├── README.md              # This file
├── color-theory.md        # Color wheel, schemes, harmony
├── paint-consistency.md   # Thinning, mediums, ratios
├── techniques/
│   ├── basecoating.md
│   ├── washing.md
│   ├── drybrushing.md
│   ├── layering.md
│   ├── edge-highlighting.md

```

## Usage

- Load color-theory.md for scheme design
- Reference specific technique files for tutorials
- Check troubleshooting.md for problem-solving

## Skill Progression

Recommend techniques by skill level:

**Beginner:** basecoating → washing → drybrushing
**Intermediate:** layering → edge-highlighting → glazing
**Advanced:** wet-blending → NMM → OSL
