# Arbitrator Knowledge Base

Quick reference rules for fast lookups without web fetches.

## Structure

```
knowledge/
├── README.md              # This file
├── phase-sequence.md      # Complete turn structure
├── core-rules.md          # Fundamental rules
├── common-interactions.md # FAQ-level common questions
├── stratagems.md          # Core stratagem rules
└── terrain.md             # Terrain rules reference
```

## Usage

- Load phase-sequence.md for timing questions
- Reference core-rules.md for fundamental mechanics
- Check common-interactions.md for FAQ answers
- Always cite Wahapedia for official sources

## Maintenance

- Update when FAQ/errata released
- Note Balance Dataslate changes
- Cross-reference with rulings.md for past decisions
